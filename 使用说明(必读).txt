CS2交易监控 使用说明（必读）
版本：v1.35

============================================================
一、安装和启动
============================================================

1. 请先将下载的 ZIP 压缩包完整解压到一个固定文件夹。
   不要直接在压缩包预览窗口中运行软件。

2. 双击文件夹中的“CS2TradeMonitor.exe”启动软件。
   软件正常运行不需要管理员权限。

   软件会把设置、凭据、日志、缓存和更新文件放在当前软件目录的
   “user-data”文件夹中，因此当前 Windows 用户必须能够写入软件目录。

3. 请不要移动、删除或单独运行 app、resources、runtimes 文件夹中的文件。
   更新程序和其他组件会由软件自动调用。

4. 本软件只提供便携 ZIP，不提供安装器。一个物理目录只运行一个实例；如需多开，
   请复制完整软件目录，再分别启动不同目录中的“CS2TradeMonitor.exe”。

5. 复制完整目录也会复制其中的账号凭据、诊断日志和悠悠设备档案。请只复制到
   自己信任的位置。凭据通常只能由同一台电脑上的同一 Windows 用户解密；跨用户
   或跨电脑复制后可能需要重新登录，软件不会自动覆盖无法读取的原凭据。


============================================================
二、首次运行环境配置
============================================================

软件需要 Microsoft .NET 10 Desktop Runtime（x64）微软官方组件。

如果电脑中尚未安装该组件，首次运行会显示中文确认窗口：

“首次运行需要配置必要运行环境”

点击“确定”后，软件会自动下载并安装运行环境，无需手动寻找安装包。
Windows 请求管理员权限时，请选择“是”。

环境配置期间会同时显示“作者自白”页面，该页面只在首次运行时自动显示一次。
环境安装完成后，软件会自动启动并打开设置界面。

如果点击“取消”，不会安装运行环境；下次重新打开软件时仍可再次配置。

如果配置失败，请重新打开软件再试一次。第二次仍然失败，请加入官方 QQ 群
1057043823 反馈。


============================================================
三、首次打开后的默认显示
============================================================

全新安装默认只显示以下两项：

1. QAQ 指数
2. SteamDT 指数

悠悠库存、单品监控、任务栏显示及其他功能不会默认强制开启，可在设置页面中
按自己的需要配置。


============================================================
四、SteamDT API Key
============================================================

QAQ 指数无需填写 API Key。
SteamDT 指数及部分 SteamDT 数据需要填写 SteamDT API Key。

设置路径：

大盘数据源 -> SteamDT 市场监控 -> SteamDT API Key

填写并保存后，等待软件刷新数据即可。
API Key 属于个人凭据，请不要截图公开，也不要发送给陌生人。


============================================================
五、常用设置入口
============================================================

界面设置：调整悬浮窗、任务栏显示、字体、颜色和布局。
单品监控：添加需要关注的饰品及价格提醒。
悠悠有品：配置悠悠登录、库存、报价及相关功能。
库存止损/盈：配置库存价格变化提醒。
Steam报价：查看和处理 Steam 报价相关功能。
大盘数据源：配置 QAQ、SteamDT 数据源。
大盘预警：设置指数点位和涨跌幅提醒。
更新与手机提醒：配置 CS2 更新提醒和手机通知。
吃米/亏米统计：查看交易统计和盈亏记录。
系统设置：配置开机启动、显示行为、软件更新、日志和恢复默认设置。

系统设置中的“编者的话”可以再次打开作者自白页面。


============================================================
六、软件更新
============================================================

可在“系统设置 -> 软件更新”中点击“检查更新”。

正式版本和自动更新优先使用官方 Gitee，连接失败时回退 GitHub：
https://gitee.com/cs2juece/CS2TradeMonitor/releases
https://github.com/cs2juece/CS2TradeMonitor/releases

更新不会主动读取或上传您的密码、登录凭据和个人配置。
不要从不明网站、网盘或收费渠道下载所谓修改版、整合版或破解版。

首次使用便携数据版本时请注意：

1. 软件不会读取、迁移、合并或删除旧的 LocalAppData 数据，需要重新配置和登录。
2. 旧数据仍保留在原位置，软件不会处理它。
3. 不支持降级到仍把数据写入 LocalAppData 的旧版本。
4. 程序文件损坏时，请先关闭当前目录实例，再重新解压当前版或更新版完整 ZIP；
   不要删除当前目录的“user-data”。


============================================================
七、常见问题
============================================================

1. 双击后没有反应
   请确认压缩包已经完整解压，并检查安全软件是否拦截了程序。

2. 环境配置失败
   重新打开软件，Windows 请求管理员权限时选择“是”。第二次仍然失败，请将
   问题截图发送到官方 QQ 群。

3. SteamDT 没有数据
   检查网络和 SteamDT API Key 是否正确，然后在大盘数据源页面查看状态。

4. 软件界面或数据异常
   先打开“系统设置 -> 诊断工具 -> 打开常规日志”查看常规日志。需要复现问题时，
   可确认开启“详细诊断模式”：它会记录所有模块，跨重启持续最多 48 小时，
   当前和已结束会话合计最多 200 MB，结束会话自动保留七天。

   网络请求体和响应体会经过递归脱敏；单个正文最多保留 5 MB，无法确认安全的
   内容不会保留。软件不会启用全局 First-Chance Exception 捕获，不会生成 dump、
   HAR 或屏幕截图，也不会自动上传日志。

   复现后可选择“打开详细日志”直接查看，或点击“导出诊断包”生成普通 ZIP。
   请先检查内容，再自行决定是否发送。页面不提供手动清除日志入口。

5. 想重新配置软件
   可使用“系统设置 -> 诊断工具 -> 恢复默认设置”。该操作会清除本机设置，
   点击后仍会再次确认。
